# Yosetta

Personal repo to learn the basics of several programming languages by building the exact same thing again, and again. In about 2000 years this repo will be super valuable mind you

## Running

```bash
$ ./node.js
$ ./python.py
$ ./ruby.rb
$ ./perl.pl
$ make cpp -s; ./cpp
$ rustc rust.rs; ./rust
$ go run go.go
```
